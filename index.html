Types of client licenses (CAL)

Client licenses (CAL) for 1 user with unlimited number of devices

Client licenses (CAL) by the number of users are suitable for your company's employees who need access to the corporate network from several different devices or if you do not know from which devices they will access.

Client licenses (CAL) for 1 device with an unlimited number of users

Client licenses (CAL) for a device are suitable for companies in which several employees can use one device, for example, when working in several shifts.
